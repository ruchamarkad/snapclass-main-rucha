# AI Powered Attendance System

An AI-based smart attendance system that uses Face Recognition, Voice Biometrics, and QR Authentication to automate classroom attendance with high accuracy and security.

---

## Features

- AI Face Recognition Attendance  
- Sequential Voice Verification  
- QR-Based Student Enrollment  
- Real-Time Cloud Database Sync  
- Secure Biometric Authentication  
- Fast and Automated Attendance Processing  

---

## Tools, Technologies and Tech Stack

### AI and Machine Learning
- FaceRecognition  
- Dlib  
- Resemblyzer  
- Librosa  
- OpenCV  

### Frontend
- Streamlit  

### Backend
- Flask  
- Python  

### Database and Cloud
- Supabase  
- PostgreSQL

## Result

- Reduced manual attendance effort  
- Improved attendance accuracy  
- Prevented proxy attendance  
- Faster classroom management  
- Secure and reliable authentication
